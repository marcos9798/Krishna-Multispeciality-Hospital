# Krishna Multispeciality Hospital & Trauma Care Website

A production-ready Next.js starter implementing a premium, conversion-focused, 3D interactive healthcare landing experience.

## Stack
- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- Three.js via `@react-three/fiber` + `@react-three/drei`

## Included experiences
- 3D hero scene with animated hospital model and ambient camera movement
- Sticky emergency CTA with urgent pulse treatment
- Interactive department cards with parallax hover motion
- Patient journey visual pipeline
- Appointment form scaffold with key fields

## Run locally
```bash
npm install
npm run dev
```
